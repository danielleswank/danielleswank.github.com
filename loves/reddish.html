So what is this project?

Reddi.sh is an open-source web GUI for the redis datastore. It was created to replace the redis CLI with a modern visual interface. The app is intended for people learning redis and the command line app interface neatly bridges the gap between a click based GUI and a console. I'm particularly happy with the UX of the command line at the bottom of the app.

Ok cool, so what did you actually do?

  • Designed the look and feel of the app.
  • UX design of the smart command line at the bottom of the app.
  • Created the logo.
  • Various collateral marketing pieces (e.g. emails).
