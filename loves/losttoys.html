Lost Toys

Lots and lots of playtesting...

lead engineer / game designer / project manager

So what is this project?

Lost Toys is a mobile game that I've been working on since a train ride back from Portland in October 2012. It's the result of hours and hours of playtesting, code arguments and ui revisions. I'm incredibly proud of it. Also here's a bit more about my studio: Sweat, Swearing & Tears

Ok cool, so what did you actually do?

  • Wrote over 50% of the C# code on the game including the rotation algorithm and the 2d screen to 3d game world controls using unity and sublime.
  • Designed multiple game levels and adjusted the difficulty curve based on user feedback and play sessions.
  • Created announcement trailer, release trailer and awards trailer using final cut pro.
  • Designed and implemented 2d marketing materials (business cards, postcards, website exc...) using photoshop and illustrator.
  • A bunch of other stuff that I've forgotten atm. Basically if it needed to get done, I had to do it including managing multiple distributed contractors.
